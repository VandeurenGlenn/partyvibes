## Development
### Get source code
```sh
git clone https://github.com/vandeurenglenn/partyvibes
```

#### Install dependencies
```sh
npm install
```
#### build/serve commands
##### build
```sh
npm run build
```

##### start
```sh
npm run start
```

##### compile
Compile js only
```sh
npm run compile
```
